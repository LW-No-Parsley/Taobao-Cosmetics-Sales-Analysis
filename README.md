# 项目介绍
本项目是一个关于淘宝化妆品销售分析的应用，旨在通过数据挖掘和可视化来研究化妆品的市场趋势。

# 特性
- 数据清理
- 数据分析
- 可视化展示

# 技术栈
- Python
- Pandas
- Matplotlib
- Seaborn

# 项目结构
```
LW-No-Parsley/Taobao-Cosmetics-Sales-Analysis
│
├── Data_clean.py       # 数据清理模块
├── main.py             # 项目主入口
├── data_test.py        # 数据测试模块
└── requirements.txt     # 项目依赖
```

# 安装
1. 确保您已安装Python 3.x。
2. 克隆该仓库：
   ```
   git clone https://github.com/LW-No-Parsley/Taobao-Cosmetics-Sales-Analysis
   ```
3. 安装所需依赖：
   ```
   pip install -r requirements.txt
   ```

# 使用
运行主程序以启动项目：
```bash
python main.py
```

# 数据分析维度
- 销售额
- 产品类别
- 消费者年龄
- 销售地区

# 数据字段
- `product_id`: 产品唯一标识符
- `product_name`: 产品名称
- `sales`: 销售量
- `category`: 产品类别
- `price`: 产品价格
